# Handshake Backend

## Setup

### Install dependencies

```bash
npm install
```

### Generate Prisma Client

```bash
npx prisma generate
```

### Create .env file

```bash
touch .env
```

### Add the following to .env

```bash
PORT=5001
DATABASE_URL="YOUR MONGO URI GOES HERE"
```

### Start the server

```bash
npm run start
```
